# OpenAI Text2Image API Guide

This guide provides instructions for setting up and using the OpenAI Text2Image API node in nndeploy.

---

## Prerequisites

Using the OpenAI Text2Image API requires an active OpenAI account and API key.

---

## Setup Instructions

### Step 1: Obtain OpenAI API Key

1. Visit [OpenAI Platform](https://platform.openai.com/)
2. Sign up or log in to your account
3. Navigate to API Keys section
4. Create a new API key
5. Copy and securely store your API key

### Step 2: Configure API Key in Node

1. Open the OpenAI Text2Image node configuration
2. Locate the API key parameter field
3. Paste your OpenAI API key into the designated field
4. Save the configuration

### Step 3: Verify Setup

1. Test the node with a simple text prompt
2. Ensure the API connection is working properly
3. Check that images are being generated successfully

---

## Important Notes

- Keep your API key secure and do not share it publicly
- Monitor your API usage to avoid unexpected charges
- Refer to OpenAI's pricing page for current rates
- Ensure you have sufficient credits in your OpenAI account
